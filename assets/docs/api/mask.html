API reference for Angular WeUI mask

import { MaskModule } from 'ngx-weui/mask';

Components & Directives

MaskComponent

selector: weui-mask

Properties

NameTypeDefaultDescription
[backdrop]

点击是否允许关闭（默认：false）

[placement]

内容方向（默认：vertical）

  • top: 顶部居中
  • bottom: 底部居中
  • vertical-left: 垂直居左
  • vertical: 垂直居中
  • vertical-right: 垂直居右
  • none: 无
[bg]

内容背景色（默认：无）

[loading]

内容为Loading效果（默认：false）

(close)

关闭回调

Methods

show

显示，并返回一个Observable

hide

隐藏

Parameters

is_backdrop

boolean = false

是否手动点击关闭（默认：false）
