API reference for Angular WeUI accordion

import { AccordionModule } from 'ngx-weui/accordion';

Components & Directives

AccordionPanelComponent

selector: weui-accordion-panel

Properties

NameTypeDefaultDescription
[disabled]

是否禁止

[active]

是否展开

accordion

AccordionComponent

selector: weui-accordion

Properties

NameTypeDefaultDescription
[collapsible]
false

是否可折叠，true 表示同时所有都允许展开，false 表示同时只允许一个展开

[activeFirst]
true

自动展开第一次

(select)

展开时回调，参数为面板下标。

Services

AccordionConfig

可用于[全局配置]。

Properties

NameTypeDefaultDescription
collapsiblefalse

是否可折叠，false 表示保持只有一个可折叠，true 表示所有都允许

activeFirsttrue

自动展开第一次

animate

动画类型，none 无动画，slide 滑动
