API reference for Angular WeUI form

import { FormModule } from 'ngx-weui/form';

Components & Directives

InputDirective

文本框，指令是对文本框格式校验（邮箱、手机、身份证等）、视觉效果的增强而已

selector: [weui-input]

Properties

NameTypeDefaultDescription
[weui-input]

文本框类型，不等同于的值 因为 weui-input 属性只是内置格式校验的简洁写法而已。 内置包括：number/digit(允许小数点)/qq/email/tel/mobile/idcard，如需要更为复杂的校验可以使用 weui-regex

[weui-regex]

格式校验正则表达式，优先级高于 [weui-input]。

[weui-required]

是否必填项，等同于的值，当值必填时会有视觉效果

[weui-cleaner]

是否自动清除内容中的空格
