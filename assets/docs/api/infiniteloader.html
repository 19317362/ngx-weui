InfiniteLoaderConfig

可用于[全局配置]。

Properties

NameTypeDefaultDescription
height

height for the container, use string like '10px', default for '100vh'

percent75

滚动至x%时触发加载，默认：75%

loading

加载中文本（支持HTML），默认：weui-loadmore 组件

finished

完成所有数据加载文本（支持HTML），默认：weui-loadmore 组件

throttle

滚动节流时长（单位：ms）
