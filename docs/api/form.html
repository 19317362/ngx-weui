API reference for Angular WeUI form

import { FormModule } from 'ngx-weui/form';

Directives

InputDirective

文本框，指令是对文本框格式校验（邮箱、手机、身份证等）、视觉效果的增强而已

selector: [weui-input]
Properties
Name Type Default Description
[weui-input]
string

文本框类型，不等同于 的值 因为 weui-input 属性只是内置格式校验的简洁写法而已。 内置包括：number/digit(允许小数点)/qq/email/tel/mobile/idcard，如需要更为复杂的校验可以使用 weui-regex
[weui-regex]
RegExp | string

格式校验正则表达式，优先级高于 [weui-input]。
[weui-required]
'info' | 'warn' | 'waiting'

是否必填项，等同于 的值，当值必填时会有视觉效果
[weui-cleaner]
boolean

是否自动清除内容中的空格

VCodeDirective

获取验证码

selector: [weui-vcode]
Properties
Name Type Default Description
[weui-vcode]
Function

发送事件
[weui-seconds]
number 60

时长（单位：秒）
[weui-tpl]
string

倒计时模板，使用 ${num} 表示当前秒数
[weui-error]
string

重新发送提醒文本

TextareaDirective

文本域字数统计

selector: [weui-textarea]
Properties
Name Type Default Description
[maxlength]
number

最大长度
[weui-cn]
number 1

中文部分应该算多少个字符，使用【/[^\x00-\xff]/g】正则表达式统计中文部分（默认：1个字符）
