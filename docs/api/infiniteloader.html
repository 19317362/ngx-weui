API reference for Angular WeUI infiniteloader

import { InfiniteLoaderModule } from 'ngx-weui/infiniteloader';

Directives

InfiniteLoaderComponent

selector: weui-infiniteloader
Properties
Name Type Default Description
[config]
InfiniteLoaderConfig

配置项
(loadmore)
EventEmitter

加载更多回调
Methods
resolveLoading

设置本次加载完成
setFinished

设置结束

Services

InfiniteLoaderConfig

可用于[全局配置]。

Properties
Name Type Default Description
height string

height for the container, use string like '10px', default for '100vh'
percent number 75

滚动至x%时触发加载，默认：75%
loading string

加载中文本（支持HTML），默认：weui-loadmore 组件
finished string

完成所有数据加载文本（支持HTML），默认：weui-loadmore 组件
throttle number

滚动节流时长（单位：ms）
